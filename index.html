Powerful, extensible charting in Perl!

Chart::Clicker is a Perl module for generating beautiful charts. Using Cairo it can generate charts in various formats. It also has a number of renderers for different types of charts. Please read the docs on CPAN. I also welcome submission of issues or suggestions. You are also welcome to drop me a line if you need help, want to tell me how much you love me, or whatever.
